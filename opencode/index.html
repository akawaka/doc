🎯 What is This?

This is not a framework.

It's a plug-and-play OpenCode config, tailored for Akawaka's real-world projects.

You get:

  • 🛠️ Prebuilt agents - Ready-to-use development tools
  • 📋 Templates - For specs, tasks, and documentation
  • 📚 Team-friendly documentation - Everything you need to know
  • 🧭 Clear, opinionated workflow - No guesswork required

No boilerplate. No guesswork. Just productive, reliable development.
